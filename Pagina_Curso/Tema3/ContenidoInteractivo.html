Contenido Interactivo



HTML ofrece una selección de elementos que pueden ayudar a crear objetos de interfaz de usuario interactivos.

Details y Summary

El elemento HTML Details es usado como un widget de revelación a través del cual el usuario puede obtener información adicional.

El contenido del elemento summary puede ser cualquier encabezado, texto sin formato o HTML que se pueda usar dentro de un párrafo.

Dialog

El elemento HTML dialog representa una caja de diálogo u otro componente interactivo, como inspector o ventana.

Summary

El elemento HTML summary especifica un resumen, título o leyenda para el cuadro de divulgación de un elemento details. Al hacer clic en el elemento summary, el estado del elemento padre details cambia entre abierto y cerrado.

EJEMPLO

I have keys but no doors. I have space but no room. You can enter but can’t leave. What am I? A keyboard.
